假期之后，开学之前

发表于 2015-01-11 | 更新于 2015-07-08 | 分类于 Memory | |

新学期开始前的碎碎念

开学前的最后一晚，有点像拿着护照通行证过关，一边是假期，一边是学期，而留给你改变的时间，是一个晚上。假期的懒散一如既往，即使收拾好了房间，却没有收拾好心情。直到跑完步之后，才迈出了新学期的第一步。

新的校区，新的宿舍，新的生活，唯一不变的可能就是跑步了。说是不变，也有变化，虽然还是“队长”，跑步的内容却不再只是一路领跑，还需要观察大家的姿势，告诉他们如何呼吸以最大效率提高给氧量，示范如何跑动以最小化对膝盖的冲击。其实这也挺有意思的，尤其在是学校边公园里的环湖跑道，也许是身体上迈出了步伐，心理上才能真正变化吧。

假期里做的最疯狂的事情，就是各式各样的聚会：同学朋友陌生人，从一二十年交情到萍水相逢一面之缘，想让自己能以更加开放的姿态去面对各式各样的人，哪怕被丢到一个完全陌生的环境，也不至于走投无路。

事实证明，也没有很难。有些事情硬着头皮去做，做着做着，头皮也就软下来了。自然而然笑与闹，只要诚心相待，结果总不会太糟。

忽然想起这段话：

当你老了，回顾一生，就会发觉：什么时候出国读书、什么时候决定做第一份职业、何时选定了对象而恋爱、什么时候结婚，其实都是命运的巨变。只是当时站在三岔路口，眼见风云千樯，你做出抉择的那一日，在日记上，相当沉闷和平凡，当时还以为是生命中普通的一天。但一场巨变，已经发生了。地动山移，浑然不觉，当时只道是寻常。世上的生死荣衰，不就是在空寂之中缘起缘灭的吗？

新的一年，学会顺势而为，缘深缘浅，笑笑也就溜过去了。只是想要的缘，去争取；不想做的孽，就躲开。

应该能好好睡一觉了吧。
