告别 LOFTER，开启独立博客纪元

我要离开 LOFTER 了，从一开始写给自己看的闲言碎语到不知怎么搞的有了十来万粉丝，虽然比不上各路摄影师插画师，但作为一个码字儿的，在这个大家都不怎么看书的时代，能遇到这么多有缘人，也是大大的幸运。

大概也算的上是 LOFTER 比较早的一批用户了，转眼两年过去，我成长了，产品本身也成长了。可是我们的方向好像有些不同，现在的 LOFTER 越来越像是一个图片社区（当然可能一直就是），留给文字的地方，不多了。

可是我只是想找一个地方，能安安静静写写字儿，记录一下日常的生活和感悟，于是有了这个独立博客。

在这里，无论是电脑和手机访问起来都更加清爽，没有其他干扰，让我们回归到生活与文字的本身。我也搬运整理了大部分自己觉得比较好的日志来这里，也算是个精选集吧。

两年写了六百多篇也是大大出乎自己的意料，是时候有一个新的开始了。

PS. 这个新的独立博客有一些特性和 LOFTER 不同，这里说明一下：

  1. 这是一个纯静态的博客，部署在 github 上，如果国内访问比较慢的话，也可以方便放在新浪/百度/阿里的服务器上，不过我在家测试的话感觉速度还可以（然而我的确做了新浪的备份，可以通过 wdxtub.cn 来访问）
  2. 因为是静态博客，所以评论系统是外挂的，外挂的意思是要用微博之类的账号登陆一下即可，但是也由于这个原因，可能回复会慢一点，不过放心，都是会回复的
  3. 另外这里可以加一些友情链接，同样喜欢写作的小伙伴可以互联一下
  4. 因为去掉了搜索功能，所以可以用标签来进行检索，打开标签的方法是把鼠标放到个人介绍下面的小图标上，然后就可以选择标签
  5. 新博客的点点滴滴都还在摸索，如果大家感兴趣，会大概写写怎么搭建自己的独立博客的
