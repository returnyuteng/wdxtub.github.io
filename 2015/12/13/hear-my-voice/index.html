直到眼睛，捕捉到光的缝隙，还可以惊天动地。

正文

下周才是这个学期的终点线，可是考试结束之后就自动进入了放假模式——起得更早了。从前大冬天不到七点我是万万没办法起床的，最近起床时间被提早了一到两个小时不等，也许没有太大的学业压力，就又切换回自己习惯的作息模式了。

不由得回想起大四没啥事儿做赋闲在家的日子，从饮食到作息，都非常简单纯粹，可谓『两耳不闻学校事，一心只读课外书』。现在才慢慢意识到，一开始是我塑造习惯，现在是习惯推动着我。就拿读书和写作来说，从积极参加多看阅读的每周读书会，到开始尝试书评以外的文字，一点一点能感受到心智的补完。很感谢多看用赠书的方式激励着刚开始塑造习惯的我，虽然现在因为一些客观原因远离了原来的圈子，但是认识的一帮各行各业的朋友，都让我获益匪浅。如今，也深深意识到读书写作对自己的推动，更想要去刨根问底，更努力去探索发现，更辩证去思考问题，更坚定去接受挑战。我甚至不再担心找不找得到心仪的工作，反而变得像夸父一样，为了看似『遥不可及』的梦想，拼了命想要赌一把。

来了 CMU 才知道 Java 的势力有多么强大，虽然到现在我依然是坚定的 Java 黑，或者勉强黑转路人吧。Android 开发这门课非常有培训班的感觉：老师坑爹助教还算靠谱但修行还真的是靠个人。唯一比较有趣的部分是队友弄错了项目展示的时间，直到还剩两个小时就要展示的时候我们才知道，饭都来不及吃就开始了一场 Death Race。几年前比赛时候还试过展示前十五分钟才把代码写好的绝杀，所以面对这种没那么『千钧一发』的情况，还是游刃有余的。虽然感觉助教给了很多组满分，但是至少我们也是满分，我觉得就足够了。

机器学习课程空有 ML 这么一个多重含义的缩写，学习过程中也的确跑偏了。本以为会更接近于机器学习实战，现实却是机器学习推导。当然这也有好处，至少把线性代数高等数学概率统计等数学基础的技术债都还了，不再害怕又长又臭的公式，甚至还能自己推导出一些结论了。学习过程有多挣扎，新生之后就有多顽强。忽然想到了 EVA 老剧场版的标题『死或新生』，可能我就是多了那么一点点韧性，一遍不懂两遍，两遍不懂三遍。抵抗力最大的方向，其实也就是梯度最大的方向，也就是说，沿着这个方向进步最快。

最后一门是算法课，实话说，不是因为找工作需要刷题，刷题需要练手，我是万万不愿意上的。因为我觉得算法是一种理解和感悟的过程，当然我不是否认肌肉训练的重要性，只是这种填鸭式题海战术，从长远角度来看究竟有多少意义？更抽象一点说，编程本身一种异化的过程，所谓异化，是让自己按照计算机的工作方式来思考，而刷题可谓是异化的比较级与最高级。对于别人可能没什么，我这种理科生中的文科生在写代码的时候就明显能感受到自我冲突，浪漫与现实，随性与逻辑，发散与收敛。在算法这块，我想按照自己的想法去学习，而不是一周学一门语言其实啥都没学到。

无论知不知道自己想要什么，烦恼总是会有的。有的人烦恼找不到自己喜欢的事情，有的人烦恼有自己喜欢的事情却被迫做更多不喜欢的事情，我属于后者。当然我觉得解决的方法也很简单，让自己变得足够强，能自由选择自己想做的事情。

直到我，波澜不惊的生命，还可以惊天动地。
