iOS 上 Markdown 编辑器简评

发表于 2015-12-08 | 更新于 2015-12-09 | 分类于 Technique | |

Markdown 作为我日常工作学习的记录语言，其高效快捷灵活给我带来了很大的便利，电脑端对于 markdown 的支持非常好，有很多好的工具软件，比如 MWeb，而且大部分支持插件的代码编辑器都可以很好的支持 markdown。但是在移动端，尤其是 iOS 上，一款符合我要求的编辑器一直没有出现，这里简评以下我能搜到的各个 markdown 应用，给大家一个参考。

先来说说我的要求，基本上一个符合我使用习惯的笔记应用需要满足以下几点：

  1. 扩展 Markdown 语法的支持，比如 gfm
  2. 网盘文件云同步，这样无缝链接各个设备
  3. 预览渲染效果美观
  4. 方便分享或者转成图片

差不多就是这些，看起来也没有很高的要求嘛，那么我们来看看今天参加评选的应用们都长啥样吧。

我的选择

1Writer

已经找不到足够的词来形容这个 app，要吹我可以吹一夜，所以就随便说几点

  • 支持包括 Dropbox 在内的四个第三方同步平台，和电脑上的文件完全无缝对接，甚至可以做到这一秒在手机上编辑，下一秒电脑上的内容也会更新
  • 字体样式什么鬼几乎都可以设置，编辑器功能齐全，字数统计快捷键盘甚至 gfm 待办事项也内建了支持（可点击）
  • 预览功能中规中矩不好看但是也不难看
  • 应用可以选夜间主题黑底白字逼格爆表
  • 支持固定到通知中心勉强算是弥补了一下 iOS 没有小部件的遗憾
  • 更拽的是，支持 URL 和 Javascript 自定义操作，也就是说，只要功夫深，可以搞出各种扩展功能（我没时间折腾就是了，其实可以内置一些给用户选）

总的来说，这是一个向往自由的人用的 markdown 笔记应用，自由选择文件格式，自由选择编辑方式，自由选择同步位置，自由选择分享方式，一切都可以按照你喜欢的方式去做，而不用受平台或者帐号的限制，可能在移动设备不明显，但是当这种自由和其他的自由以简单和谐的方式配对起来的时候，就会发现其强大威力（以后会详细介绍我的知识积累体系）

值得一试

  • 锤子便签
    • 拟物设计看着那个别扭
    • 对 markdown 支持那个不完全
    • 一定要登录，所以第三方同步也是不用想
    • 为什么我在用呢，就因为能方便生成图片分享到微博
    • 为什么我在用呢，因为 flyme 的笔记还没有搬到 iOS 上
  • Day One
    • 已经不知道被苹果推荐过多少次的应用，支持 iCloud / Dropbox 备份，但是没办法编辑自己的东西
    • 能想到的功能基本都有，图片文字标签日历加星导出 pdf 十全大补丸
    • 怎么说，文艺小清新值得一试，跨平台客户端也非常好用，iOS 版有时限免，Mac 版非常坚挺
    • 记日记的话，这几乎是最佳选择
  • Clarity
    • 支持 Dropbox 与本地存储，但不支持和 Dropbox 已有的文件同步
    • 预览非常美观大气，基本的格式也都没有问题，真的漂亮，加分
    • 可以置顶特定的笔记，手势操作设计也很合理
    • 编辑器软键盘有 markdown 符号快捷键，非常好用
    • 如果是从新开始建立笔记，这是一个不错的选择

有特点也有硬伤

  • Evermark
    • 曾经非常好用的编辑器，为什么说是曾经，因为很久没有更新，在 6s 上强制分辨率的缘故，字体发虚
    • 编辑器美观实用冠绝全场，作为一个轻应用这个特点非常吸引人
    • 预览效果非常赞，完美支持 gfm，好看好用
    • 不支持同步，作为本机的小笔记本非常靠谱
  • Leanote
    • 开源，云同步，所以不支持第三方网盘，扩展性一般
    • 编辑器是富文本的，用的时候爽但是兼容性恐怕难以保证
    • 需要登录才能使用，但是各种功能都很全，算是很好的印象笔记类重型应用
    • 支持自建服务器，隐私性可以保证
    • 可以顺手发布博文，不过似乎用的人不多
    • 总体来说感觉什么都做了但是什么都没有做得特别好，尤其是图标，三个字：难看
  • Listacular
    • 可以和 Dropbox 连接，支持各种格式甚至待办事项这种扩展，手势操作也没有落下，功能还是颇有心意的
    • 免费版有使用限制，只能创建三条笔记
    • 没办法分享，只适合零散的记录
    • 根据个人需求来决定用不用
  • Beauteous
    • 可以导出为 pdf，但是不支持 Dropbox 同步
    • 预览效果不错，基本的格式没有问题
    • 字体及大小都可以自由定制
    • 列表条目展示设计有些怪异，题目靠右，内容靠左，看起来比较奇怪
    • 不支持新建分类
  • WriteBox
    • 支持同步比较多 Dropbox, Google Drive 或者本地都可以
    • 预览效果中规中矩，也可以自定格式，但是编辑器的快捷键设计有些鸡肋，界面不够扁平然后就放零星几个按键
    • 总体来说没什么设计感，一个字：难看
    • 虽然不至于不能用，但是硬伤也是挺伤的

不建议使用

  • Wright / Write.r / Wryte
    • 把这三个放在一起是因为，它们是互为山寨的，给我的感觉是找了个开源项目改改就拿出来卖了，目的基本是捆绑推广其他的应用，差评
  • WriteRight
    • 连我基本的格式测试文本都没有办法正常渲染，基本功能缺失，差评
  • Edhita
    • 奇怪名字，显眼处加了广告(因为是免费版)，完全不能忍，差评

总结

1Writer! 1Writer! 1Writer!

我先干为敬，大家随意，就是这个意思。
