我就是我

发表于 2014-02-13 | 更新于 2015-07-08 | 分类于 Memory | |

选择了另一条路，就不要回头

新的一年刚开始，人生轨迹似乎就发生了意想不到的转变，看了很多，想了很多，也终于做了决定。

所以我想记录下来，也好给几十年后的自己留个念想，假如在将来真的能回到过去，我想告诉现在的自己，你做了一个不错的决定。

事情是这样的，当初放弃留在微软选择读研之后，一直在想，我到底选对了没有。慢慢我想通了，其实选择本身永远是没有对错的，做选择之后怎么做才有好坏。无论做了什么选择，只要之后拼尽全力去追寻，那这个选择，就是不错的决定；而若是之后开始自怨自艾止步不前的话，那才是糟糕的决定。

所以我决定换一种方式去生活，学着独立，不再向家里要钱，抓住可以抓住的每个机会去赚钱。然后我发现不但可以养活自己，甚至还可以攒钱买点其他小玩意儿。一边上学一边赚钱可能是所有勤工俭学学生的梦想，可是看起来很美的表象下，有一个问题：当你发现其实这样我也能挣到一些钱之后，主次观念就变了，本来是通向梦想的手段，成为了梦想本身。既然现在我就能过得很舒服，为什么还要去追寻那虚无缥缈前途未定的路呢？

年轻时经济上的过分优越反而可能害了你。特别是当这份优越的收入并非来自你梦想的职业时，金钱上的优越会麻痹你应有的人生觉悟和冲劲，让你变得现实与功利，失去了原始的梦想，不知道自己该干什么，未来也会越来越迷茫。

只有坦然地接受经济上的窘境，不要为此过分苦恼，并将其化作动力，你将来才不会后悔自己的选择。 你们还年轻，人生之路何其漫长，不要将眼前的利益作为所有事物价值和意义的评价标准。在做事时，三思而行，要想想，这件事会把我塑造成什么样的人，对我的人生有没有帮助，这才是比钱更值得考量的评价标准。 比金钱更重要的是未来。

年三十的下午，导师问我愿不愿意参加一个和CMU合作办学的项目，这意味着我需要在14年的夏天重新入学，再读两年。只是比原计划多读一年而已，纳闷自己在纠结什么，明明是一件好事不是么？

忽然我明白了，我是在害怕，害怕逃离舒适区。

才发现，我像温水里的青蛙，很舒服很满意：有许多玩得来的朋友，有值得努力的方向，有看中自己的长辈，有养活自己的工作，一切看起来都很美好，确定且美好，我只需要按照这样的步伐大步向前即可，升职加薪，出任总经理，当上CEO，迎娶白富美，走向人生巅峰。

但是忽然要放弃这确定的一切，去拥抱未知，未知的环境，未知的将来，未知的命运，可悲的是人的本能恰恰是讨厌不确定的。一面觉得这是一个很好的机会，一面担心为此放弃现在的生活是否值得；一面讨厌不敢接受挑战的自己，一面害怕挑战最后得到的反而不如从前。

才意识到，逃离舒适区，需要多么大的勇气，来做多么大的决定。

和兄弟们喝酒的时候谈起这事，他们都鼓励我要像个男人一样，勇敢接受挑战。其实那时候我就默默决定，无论如何不能给兄弟们丢脸，咱们军人的子弟，绝不能做怂人躲在角落，只有要战就战战个痛快，没有其他的！

今天收到了录取通知书，心里却没有想象中激动，可能是因为之前预想过太多次了吧。选择了远方，也就选择了更加自由，却又更加辛苦的生活。可是既然之前可以做的不错，为什么不能相信自己这一次同样也可以呢！

现实和理想之间总会有差异，这差异让我们遗憾或悲伤，但是更可以说这种差异是好事，因为当回首人生的时候会发现，也有那说走就走的旅行，和说闯就闯的九州。

请相信自己的潜力。就算人生目标还不明确，又或信心产生动摇，都请充分相信自己。不要将眼光局限在过于功利的考试、资格证书、学分等事物上，而将无限的可能性扼杀在摇篮中。 请你暂时停下脚步，为自己缜密的计划留一些余地，偶尔期待一下偶然。暂时将自己托付给时间，等待上苍新的安排，也许会比埋头奔跑取得更大的成就。我就是我，难道不能按这种逻辑信自己一回吗？
