坐标系

发表于 2014-03-04 | 更新于 2015-07-08 | 分类于 Thinking | |
960 | 3

你有没有属于自己的坐标系？

昨天晚些时候，和多看的媛媛聊了聊，她说看我的书评，大多是为读书会的书籍写的，但是若是想把我放到更好的位置，就需要让我多写写其他的书，最好有自己喜欢和擅长的书籍类型。具体要怎么做呢，找到一个方向，持续地精读。

一语点醒梦中人，我这才意识到，虽然读了不少书，但是点开归档就会发现，刨去数量不看，杂乱无章法，没有整体的思路和逻辑。要问我最喜欢什么类型的书，我答不上；要问我最擅长什么类型的书，我还是答不上。忽然很惶恐，原来看了半天书，居然连什么是自己喜欢的都不知道。就像在走迷宫，有的人一边走一边记录，确定之后的方向，而我却是走到哪儿算哪儿，没有一个特定的目标，仅仅去关注一路的风景。不是说这样不好，只是我觉得，也许是时候需要一个方向了，这样我至少知道自己是在一路前行。

点开书城的分类页面，我试图从中寻找方向，分类很多，第一次筛选出来的类型有（类型后面的数字为书籍数量）：

小说（2675），两性（173），经管（1041）， 计算机（136）， 科学与自然（188），文学（1215），生活（889），传记（612），旅游（186），历史（452），政治与军事（203），艺术（178），社会科学（853），教育（146），外语（131）。

小说这个分类太宽泛，更多是休闲用，基本上大家都会喜欢，在这种类型上想要擅长，很难。两性的话我这样一个毛头小子走过的路还没别人吃过的盐多，想想还是算了，是要慢慢学的。经管倒是我比较感兴趣的一类，之前PM实习的时候也接触了一些，我觉得靠谱，可以培养自己独特的观察视角。计算机不用说，只是多看目前这类书还是太少了，而且一般看书周期比较长，要慢慢沉淀。科学与自然，这个类型也不错，理科生出身对于物理化学地理总是有割舍不掉的情结。文学，这个水太深，活到老学到老。生活，可能要真正有了自己的生活，才有可能擅长吧，也只能跳过了。传记，这也是我非常喜欢的类型之一。旅游，虽然喜欢，可是很长一段时间内都还要在学校度过，等我真正开始行走江湖，再专攻这部分。历史，业余爱好者，也只能看看了。政治与军事，一般一般，可能年龄不够。艺术，我很在意的一个领域。社会科学，涉及太广，高山仰止。教育，还在接受教育中。外语，这部分书城的书籍质量一般。

所以第二次筛选出来的就剩下：经管、计算机、科学与自然、传记与艺术了。但是这显然也太多了，学业一忙，就很难兼顾了，于是还要再筛选。不得不做出许多艰难的决定：经管的毕竟涉及面广，我阅历也太少。计算机的话我研究的只是非常小的一个子集，何况书城上还没有这个方向的，也只能作罢了。传记的话，似乎很难去说怎么样是擅长，更多的还是看得多了解的多。

最后就剩下科学与自然，还有艺术了！而艺术是曾经想过但一直没有去涉足的领域，就这样决定了！从艺术开始吧！
