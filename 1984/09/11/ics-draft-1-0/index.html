【计算机系统导论】第一章 群星之光 模仿游戏

发表于 1984-09-11 | 更新于 2016-10-13 | 分类于 Technique | |

欢迎阅读本书！让我们一起在计算机系统的世界中开启一段神秘且有趣的探险之旅吧。

欢迎阅读本书！让我们一起在计算机系统的世界中开启一段神秘且有趣的探险之旅吧。

无论从设计还是工程的角度来看，计算机系统都算得上是人类智慧的精华。复杂的细节隐藏在简洁的抽象之下，与其说计算机是一种设备，不如认为它是『计算思维』的具象化。

那么什么是计算机系统呢？正所谓横看成岭侧成峰，远近高低各不同，我们也很难给出一个明确的定义。在本书中，我们认为计算机系统是由硬件、操作系统、应用程序和用户这四部分组成的。

『硬件』提供基本的计算资源，比如我们熟知的中央处理器、内存和输入输出装置，是计算机系统的骨骼。『操作系统』在不同的用户与应用程序中控制和协调硬件，是计算机系统的神经系统。『应用程序』使用系统资源来计算解决用户的问题（数据库、游戏、商业应用等），是计算机系统的血肉。『用户』则可以是人、机器或者其他计算机，是计算机系统的各种感觉，也可以看做是整个机体的高层表现。

在谈论计算机系统的时候，我们常常把操作系统和应用程序划分为软件这以大类，刚好和硬件相对应，却常常忽视用户这一类。但换个角度看，我们在学习计算机系统的时候，更应该通过对硬件和软件的理解，加强我们作为『用户』的能力，真正利用好计算机系统来发现问题和解决问题。

在计算机系统的世界探险的过程中，我们会逐渐学会如何通过抽象和分解，用合适的建模方法来表示复杂问题，将其转化为已有解决方案的问题，并最终解决它；我们会逐渐意识到计算机系统的组织和设计无不是从现实世界中获取的灵感，和很多看起来毫不相关的事物都有异曲同工之妙，比方说我们常用的时分秒计时，就和计算机的存储系统有着同样的分层设计。

最终到达这次旅途的终点时，我们就会发现计算机系统的本质是抽象和自动化，便可以用『计算思维』站在计算机的角度来理解和思考问题，才能够更好地利用计算机系统来解决问题。

万丈高楼平地起，本章中我们会从掰手指计数一路介绍到现代计算机，无论是东方的算筹，还是西方的计算尺，了解这些看起来过时的概念，其实对理解计算机系统很有帮助。图灵机作为计算机科学中非常重要的理论，也会简单进行介绍，并据此给出计算机能解决的问题的明确定义。然后会简单介绍硬件系统和软件系统的关键概念，以及计算机是如何结合这两个系统来解决具体问题。最后会探讨『计算思维』中几个非常重要的理念与思想作为收尾。

计算机科学是一门实践科学，计算机系统也是经过一代代人不断尝试和改进才发展到今天这种规模。很多现在看起来司空见惯的概念，在提出的时候很可能被认为是天方夜谭。随着硬件设计的完善和制造水平的提高，计算机的运算能力早已远远超过人类，但人类的最大优势是想象力。一百多年前，计算机科学的先驱们凭着天才的灵感和不懈努力在极其简陋的条件下完成了一个又一个壮举，操作计算机的方式也从最原始的拨轮到纸带再到现在百花齐放的交互方式。

纸上得来终觉浅，希望通过本章的介绍，能够在大家心中埋下一颗敢于异想天开和勇于尝试的的种子。跟着别人走永远没有办法领先，只有亲身踏入无尽荒原，才能在不断地学习和实践中摸索出属于自己的路。

本章的最后给出了一些计算机与社会相关的思考题，技术虽然是中立的，但是使用技术的我们却不是。计算机系统是非常强大的，当我们拥有了计算机系统的力量，一个必须要思考的问题是我们要如何去运用这股力量。希望能通过这些思考题，帮助大家把思维拓宽，把自己放到一个更为广阔的空间中去思考，也许就很能大成都上避免一叶障目的问题。

好了，我们背上行囊，准备出发吧！
