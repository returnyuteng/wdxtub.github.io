第三十六周 - 追光

发表于 2017-02-17 | 更新于 2017-02-17 | 分类于 Gossip | |

你在听吗，我莽撞的故事，和你的一样。我们一样，不顾一切奔跑，不在乎受伤。

更新历史

  • 2017.02.17: 完成初稿

离职后的第一周，状态总算是随着天气的变暖而逐渐回升。虽然这一周大部分的时间都坐在电脑前码字儿，但时不时出去走走拍拍见见朋友，也倒是颇为自得其乐，整个人的气色也好了很多。

难得赋闲在家，第一次有了『采菊东篱下，悠然见南山』的感觉。现在回过头来想想，工作上的大部分事情，我不做，也总是能找到人来做的，区别可能就是我能做得更快更好一些，但是，真的放到全局来看，这一点点差别其实无关痛痒。重要的是方向，方向对了，再傻的团队，也能摸索着找到终点，怕就怕朝三暮四朝令夕改，时间都在变来变去中蹉跎了，反而啥都没搞成。

终于找到了比较大段的空闲时间，写书的进程也大大加快了，本周重新修订了章节目录、完成了材料的筛选、整理出了初稿还要初稿的初稿。实话说，我很满意，因为有了这个基本稳定的骨架，剩下的工作就不用在占用如此多的大段空白时间了，就可以一步一步化整为零精雕细琢了。写书真的像古诗里说的那样『正入万山圈子里，一山放过一山拦』，如何求新求变，如何能跟传统的课本不一样，说实话我还没有特别好的思路，仍旧需要一段时间的沉淀与思考。

不由得想起路过客厅无意间看到的电视节目，说的正是钧窑传人如何求新求变，如何知行合一的过程。有一个地方让我非常触动，大意是这样的：哪怕是钧窑的碎片，明眼人也能看出是出自哪个朝代的，为什么呢？一是因为窑变的存在，使得钧窑就是钧窑，二是各朝代都会在此基础上发展出自己的特色，而不只是仿照前朝，这就是独门的手艺了。

窑变很有意思，《景德镇陶录》有记为『窑变之器有三：二为天工，一为人巧。其由天工者，火性幻化，天然而成；其由人巧者，则工故以釉作幻色物态，直名之曰窑变，殊数见不鲜耳』。我这么一想，其实写书，尤其是教科书，同样是这个道理。作为入门的导论类书籍，大部分的知识点其实基本已经确定了，我能做的绝非『天工』，而是『人巧』。至于怎么个『巧』法，我暂时还真的没完全想好，可以师承经典，却不能囿于传统，这里有一些零碎的念头，还需要继续探索。

我一直觉得，能有这么多靠谱的朋友结伴成长，是非常幸运的事情。有的并无血缘却胜似亲生兄弟，有的潇洒大方带我认识世界，还有的追逐梦想极富行动力敢想敢做。他们的人生轨迹各自不同却精彩纷呈，却都有一个共同的特点 — 选定了方向便不再退缩犹豫。很多时候我看到他们，便会意识到原来人生还有这么多的可能，于是也想试着去真正做自己，去按照自己的想法，走自己的路。

看繁星散落在头顶，落在背着的行囊，连雨水打在我的身上，也会化成我们的衣裳。我在发声，喊出骄傲坚持，不甘心流浪。
