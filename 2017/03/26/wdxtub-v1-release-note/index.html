发布摘要

在知和行真正合一之前，我们往往很难意识到其中蕴含着的巨大能量。没有人一开始就能了解一切掌控全局，这个时候我们需要『假设性原则』，即一个事儿在实践完成之前，都必须假设自己是无知的。知道自己不知道其实是特别好的事情，这意味着自以为是刚愎自用之类的是不存在的。与其临渊羡鱼不如退而结网坐而垂钓，一个事情不去真正做一做，是不会有太多收获的，还容易被被人的言论给误导。

所谓成长，就是学会面对恐惧战胜恐惧，而战胜恐惧的唯一办法就动起来，让自己意识到想象中的恶魔不过如此。而在行动的时候，一定要有方向有目的，毕竟我们的力量有限，只有聚于一点，才能真正突破。

其实前面的这些，也正是 wdxtub.com 1.0 版本的核心思想：在 Beta 版本的基础上进行打磨持续改进，找到用力的点，再补全短板。

  1. 技术文章的写作（即『通天塔』系列）逐渐有了一个固定的套路，即通过清晰明了的例子，写其然也写其所以然。这就要求自己去深入挖掘技术细节，不能忽悠自己。
  2. 理论文章的写作（即『不周山』系列）还在慢慢探索中，在工程与理论间有一条微妙的分隔线，我想找到最适合自己的位置。
  3. 生活文章的写作（即『好望角』系列）要更加随性些，工作要带有艺术精神，生活同样要有，不把日子过得精彩点，怎么写周记呢？

这个版本的改动并不像 Beta 版本那么大，但这也正说明之前的设计比较成熟， Beta 测试做得不错，具体请参考下面的『浏览指南』和『版本注释』。

浏览指南

设计了全新的网站 favicon，现在是一个原创的标志了（如下图）。因为我属马，但是又没有天生的翅膀，所以挂上几个气球，虽然自己知道这样远不足以飞起来，但重要的是那颗飞翔的心不是吗？

更换了文章评论的服务提供商：多说 -> 网易云跟帖，可以方便的盖楼和点赞啦！（期待和大家交流）

每篇文章都增加了字数和预计阅读时间

『不周山』系列内容精简，浓缩为五个大系列。其中数据科学和云计算系列是之后的重点施工对象。

『通天塔』系列内容重构，技术实践部分统一改为『资源索引』类型，包含我的技术文章与网上的优秀资源。

最后感谢大家一直以来的支持，我们下个大版本见。

版本注释

1.0 版更新

  • Favicon 更新，采用全新的原创标志
  • 文章的评论服务提供商更新，现在是网易云跟帖
  • 技术文章汇总成统一的『资源索引』格式，方便查阅
  • 文章增加了字数统计和预计阅读时间
  • 『不周山』系列更新，专注于计算机系统、数据科学、云计算和英语
  • 『通天塔』系列更新，专注于团队建设、W.I.S.E、极客修炼、无人机和日志分析平台
  • 清除部分老旧内容（会以新系列的面貌跟大家见面），优化访问速度

Beta 版更新

  • 导航栏更新，用『不周山』『通天塔』和『好望角』取代了原来的『作品』『技术』和『生活』板块
  • 导航栏更新，原来的『关于我』改为『关于』，增加了关于博客本身内容的信息
  • 导航栏更新，原来的『书单』改为『书影音』，增加了关于电影、音乐和游戏部分的内容
  • 每篇文章除了创建时间外，增加了『更新时间』，方便大家了解文章更新状况
  • 为了配合全新的板块设计，对大部分文章内容进行了调整，之后所有的文章都会隶属于某一板块的某一系列，更加清晰
  • 书单部分由原来的表格切换成了无序列表，并根据内容进行了更加详细的分类
  • 感谢网友 keli 的建议，导航栏图标进行了更换（虽然没有找到山和塔的图标）
