【盖洛普优势识别器 2.0】发现自己

发表于 2016-05-08   |   分类于 Reading   |     |  

人不可能事事皆行，但可以人尽其才。

测试之前

本书介绍了 34 种不同的主题，通过测试简要了解自己的优势与弱势，并据此制定自我发展方案。书中提到的一个很重要的观念就是：改善劣势是一条最难行的路，发现自己的优势才是最佳出路。

最成功的人都是先从其突出的天赋着手，然后学习技能和知识，再加以练习，以此得到成倍的优势的。预测同时要管理自己的弱势，了解自身在哪些方面先天不足，就能避免走很多弯路。

上面的公式非常有价值，天赋和投入缺一不可，一定要根据自己的实际情况，来制定不同的发展方案。邯郸学步、刻舟求剑都只能是事倍功半。

了解了这些，还没买书的同学，如果只是想体验一下，可以试试看最后给出的链接。千万注意的是，不要被测试束缚住了手脚，很多东西都是因人而异的，只要是自己笃信的东西，就不要因为别人的三言两语开始怀疑自己。毕竟这类测试，要多少有多少，只是参考，不是领导。

测试之后

34 个主题如下：

成就、行动、适应、分析、统筹、信仰、统帅、沟通、竞争、关联、公平、回顾、审慎、伯乐、纪律、体谅、专注、前瞻、和谐、理念、包容、个别、搜集、思维、学习、完美、积极、交往、责任、排难、自信、追求、战略、取悦

我觉得正确的使用方法是，先大概了解一下这 34 个主题的内涵，然后在前五大优势的基础上，挑选比较可能，自己也想拥有的其他五个潜在优势，然后在生活中注意对应发展。

具体说明和建议

这里我根据这个『不大精准』的测试得到的五大优势，以及其他一些我希望自己拥有的优势来做书摘和拓展。千万注意的是，不要被测试束缚住了手脚，很多东西都是因人而异的，只要是自己笃信的东西，就不要因为别人的三言两语开始怀疑自己。毕竟这类测试，要多少有多少，只是参考，不是领导。

另外，如果需要其他方面的建议，请参考官方网站和原书。

战略

  • 透过日常琐碎，寻找前进的捷径
  • 精确判断潜在的障碍
  • 花些时间全面自省或沉思你想取得的目标，直到想到相应模式和问题为止
  • 找到一个从事重要工作并能促进你的战略思考的团队
  • 尽可能相信你的本能判断
  • 与拥有优秀『行动』天赋的人合作
  • 确保你参与到新计划或新公司的创建早期

个别

  • 不能容忍一概而论或简单归类
  • 让他人认识到真正的多元见于细微差别之中
  • 弄清你团队中每个人的长处
  • 顾及并尊重他人的好恶，并有能力做出辨别

理念

  • 你的头脑总在寻找关联
  • 寻求能发挥你想象力的工作
  • 在工作和生活中应有一些小变化
  • 交流前完成你的思考和想法
  • 天生擅长研发
  • 与拥有优秀『分析』天赋的人合作

交往

  • 你对朋友敞开心扉
  • 希望了解他们的情感、目标、恐惧和梦想
  • 真诚的关系才有价值
  • 找一个重视友谊的工作场所
  • 主动尽可能多去了解你接触的人
  • 要与朋友保持联系
  • 为家人和朋友留出时间
  • 努力与同事或团队成员建立工作之外的往来

统筹

  • 形势越是变化无常，你越能出神入化
  • 寻找复杂、充满活力且不需要循规蹈矩的环境
  • 如果需要建立团队，确保你参与其中
  • 关注团队成员的不同个性和观点

行动

  • 行动和思考并不互相排斥
  • 最好的办法是做中学
  • 作出决定、采取行动、检验结果
  • 寻找课自主决策并采取行动的工作。特别留意那些刚刚起步或扭转局面的场合
  • 辨别出组织中最具影响力的决策者

搜集

  • 应当考虑每天都能接触新信息的职业
  • 建立可以存储和轻松查找信息的体系
  • 思维开放，渴望了解一切
  • 识别你的特长领域，积极为这一领域寻找更多信息
  • 有意识地扩展词汇水平

思维

  • 你是自己最好的伴侣
  • 用日志记录想法
  • 有意识地与『思想家』建立联系
  • 安排思维活动时间，这将激发你的能量
  • 找到愿意与你讨论相同话题的人
  • 鼓励周围的人通过整理问题和参与对话来充分利用智力

排难

  • 判断故障，排除问题，起死回生
  • 不要害怕让别人知道你愿意去解决问题
  • 让自己有机会休息
  • 别人的问题让他们自己去处理
  • 不仅要解决现有问题，还要用它来预判和防止问题的发生
  • 思考有哪些方法能提高你的技能和知识

自信

  • 深知自己是有能力的人，有能力冒风险，有能力接受新的挑战，有能力提出要求，有能力履行诺言
  • 没有人能为你作决定，没有人能告诉你如何去思考
  • 寻找尚无规则建立的起步阶段
  • 让你的自信心表现出来
  • 你独立的特性可能会让你孤立无援
  • 设定远大目标

写在最后

大概浏览了一次之后，更加觉得我对职业生涯的选择是非常正确的。不了解自己，唯一能让自己感觉安全的办法，恐怕就是随大流了。可是了解了自己，不去做自己能做好的事情，是不是有点浪费呢？

最后是如何管理弱点的五个建议：

  1. 把事情做得好一点
  2. 设计一个支持系统
  3. 用你的超强天赋来盖过弱点
  4. 找一个伙伴
  5. 停止做这件事

有的时候，从失败中学到的东西更加可贵，但这些东西不是最终目的，最终目的是利用它们走向成功。
