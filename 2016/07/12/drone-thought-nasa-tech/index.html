【通天塔之聊聊无人机】肆 NASA 的技术思路

发表于 2016-07-12 | 更新于 2017-01-21 | 分类于 Technique | |

NASA 作为美国在无人机政策指定的主心骨，其研究思路和做事方法都有值得我们学习的地方。如何引导利益博弈，如何在博弈中争取最大的合力，其实比计划本身更需要智慧。

上一篇我们了解了 NASA 的总体计划，这一篇我们来细致观察一下具体计划里有什么。

技术目标

技术目标在于保证低空无人机操作的安全。为了达到这个目标，NASA 给出来了短期和长期目标。短期目标比较务实：随着民用无人机行业的快速发展，NASA 需要尽快拿出一个系统，来基本保证飞行安全。而长期目标则抽象些，原话是『满足无人机飞行日益增长的需求，与此同时保证最高的安全性、效率和通行能力』

在此基础上，除了无人机系统本身，也需要配套的可移动/永久地面站，以支持包括但不限于下列应用领域：

  • 农业应用
  • 低空雷达
  • 摄影摄像
  • 设备巡检
  • 搜索救援
  • 货物运输

端平一碗水

民用无人机硬生生把二维的交通扩展到了三维（虽然地面交通也有立交桥之类的设施，我们姑且认为其仍旧是二维的）。空间是资源，资源是财富，这么一大笔财富要怎么分，就需要考虑到各种因素了。

最重要的当然是国家和地区的安全，对于某些核心或者机密的资产，一定得严防死守。其次是与现有空域的集成，这里则需要具体地区具体应用具体分析，可以想见这里将会是各大公司争夺的重点（Google 和 Amazon 已经开始提前入场占位了）。最后才是在商业、农业上的应用。这种趋势其实在很多产业上都已经得到验证，计算机和网络都是经过这样一个实验室-军方-政府-商业公司-个人的发展历程才有今天的。

为此，NASA 要推出/推广的 UTM 系统其实是一整套完整的服务，用以保证低空安全飞行，具体包括：

  • 空域管理和地理栅栏
  • 天气和大风预报
  • 预测和管理航线
  • 禁飞区和人造建筑数据库（避免相撞）
  • 保证安全距离（空域预留，包括无人机与无人机之间通信或无人机与 UTM 系统间通讯两种方法）
  • 只允许已授权行为

技术功能

空域行为与管理

根据目前的规定，最高只能飞 500 尺（大概是 150 米），具体管理的时候，得考虑到地形的需要和应用的需要。而一片空域如果有多架无人机，到底谁是老大呢？NASA 的规则也非常简单粗暴：谁性能好谁是老大，毕竟性能好的（尤其是安全性能）更能保证自己执行任务的稳定性。

这之中一个很重要的技术就是地理栅栏(Geo-fencing)，其中又包括动态和静态两种。静态的比较好理解，比方说风景名胜、机场设施等重要的建筑（尤其是对领空有要求的），都可以提前输入到禁飞系统中；动态的则针对于突发事件，比方说火灾、爆炸、恐怖袭击等等。

大风与天气信息集成

影响飞行的天气主要有几种：大风、降雨和打雷。无人机本身没有感知天气的能力，用户在飞行的过程中可能也无法通过目测来判断接下来的天气，这样的话一旦出现影响飞行甚至是极端天气状况，可能会造成事故，所以如果可以利用 UTM 系统来告知用户相关信息，在一定程度上就增加了用户和无人机的环境感知能力。

拥堵管理

拥堵简单来说就是需求与容量的不平衡，就像城市道路交通一样，很多时候僧多粥少，可以缓解，但不可能解决。针对无人机来说，空中走廊，高度划分和方向划分都是可能的解决方案，不过这些都应该仅在必要时采用。

分隔管理

有的时候我们需要对特定空域进行隔离（保留），每架无人机在飞行中也需要和其他无人机保持安全距离。具体可以分成 V2V（无人机到无人机）或 V2UTM（无人机到 UTM 系统两种解决方案）。而追踪的方式主要有三种：基于 ADS-B，基于手机和基于卫星。不同的方法各有利弊，会在后面的文章进行介绍，这里就不展开了。

突发事件应对

这里主要是为了利用无人机来应对类似 9-11 事件的突发状况，尤其是在大规模 GPS 和手机都失效的情况下，如何才能有效调度无人机来进行应对呢？相信这也是一个很大的技术挑战。

商业模型

UTM 目前虽然由 NASA 带领开发，但最终 的商业模型仍然没有定论。可能的方式有：

  1. 单一服务提供：政府机构。类似于 FAA（Federal Aviation Administration）
  2. 单一服务提供：非政府机构。提供一个 Web 服务
  3. 多服务提供：州政府、当地政府机构。可以根据不同需要进行部署
  4. 多服务提供：非政府机构。不同公司完成不同区域的部署

这个时候我就要说还是由政府来统一组织管理比较好，至少不会像美国银行业那样撕裂。比方说匹兹堡最流行的 PNC 银行，在湾区是找不到的；而在加州很多的 Chase 银行，在匹兹堡是找不到的。听起来就很不方便，更不要说又贵又慢的转账了。所谓『集中力量办大事』，我觉得应该就是办这类事儿。
