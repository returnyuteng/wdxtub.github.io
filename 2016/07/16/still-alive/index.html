第五周 - Still Alive

发表于 2016-07-16 | 更新于 2016-07-16 | 分类于 Gossip | |

Now these points of data make a beautiful line. And I’m out of beta.

来深圳一个多月了，终于在这儿度过了第一个周末，还是每天都要去医院打打吊针的周末，果然吧，新生活不是说适应就能适应的。工作生活和学习生活真的是有蛮大的差别，整个人心态的变化也非常明显，希望借着这次的『妖风』，真正全身心投入新的战斗吧。

这周又接触了一两个新项目，无论是帮运营出文案，还是参加头脑风暴，亦或是和产品经理讨论数据挖掘思路，都在不断接触和学习到新东西，对于我这种好奇宝宝可是再开心不过了。这两天听内核恐慌早期的节目，听到消极姐说在大公司里干活的感悟，真的好有共鸣。她说做成功的项目吧，到最后可能只有 10% 是自己做的，不好意思出去说，而做不成的项目，直接就不能说。我一想，还真是这样，很多时候『不足为外人道也』也是痛苦的，好像白来一遭似的。所以又回到那句老话：一个人的命运当然要靠个人奋斗，也要考虑历史的行程。

和大洋彼岸团队合作的项目总算是在大家的努力下逐渐运转起来，因为很多都需要和美国团队沟通，所以基本上我要负责这边的项目管理、产品经理、后台以及一小部分前台的开发工作，感觉非常刺激，真的是有很多地方需要去思考和权衡，我相信这也是快速成长的最佳途径。不过正如之前说得那样，细节的东西不能说，所以看起来只能是夸夸其谈了，但是等项目做完之后应该就没问题了，期待这一天早日来临。

周五晚上在同事的带领下第一次飞了无人机，虽然之前在公司里已经见过太多，不过真正实际操作的时候，还是有很多不一样的感觉。从连接到操控，从配频到图传，虽说不如想象中那么『美』，但是真正从二维跨越到三维的时候，真的感受到了多一个自由度那种爽快。反过来想，其实读书给我的自由更多，借用《三体》中的『水滴』，知识间足够大的强相互作用力能够形成无坚不摧的实体。可惜的是，肉体太脆弱，周四的时候，病倒了。

真的是『倒』了，好在是在公司，不然在家的话，就不造咋办了。所以我发明了一个新词——空巢青年，指那些因为病假在家，但室友都去上班的上班族。躺在急诊病床上的时候，满满 EVA 既视感，好在还有贴心的同事跑上跑下照顾我，不然真的是非常无助。烧得稀里糊涂反而想到了很多东西，不过正常之后反而少了点感觉，下次应该掏出手机录个音才对。总的来说，虽然身体很难受，但是得到同事们满满的关爱，慢热的我也借着这次『妖风』活络了起来，塞翁失马嘛。

因为这一段时间都要吃清淡的缘故，周末在家试着叫了一次外卖，体验好到不敢相信，国内的 O2O 服务真的太厉害了，已经彻底被征服。买的蚊帐也到了，搭的时候真心想起了在匹兹堡拆拆搭搭帐篷的日子，特此贴图纪念一下。

明天最后一次吊针，也就算渡劫了！接下来要好好安排自己的学习工作生活，身体是本钱嘛。

再次感谢帮助我的同事朋友及医护人员，让我这个『外地人』，第一次有了『家』的感觉。

I’m releasing on time. So I’m GLaD. I got burned.
