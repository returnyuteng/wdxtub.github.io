第三周 - 在木星

发表于 2016-07-03 | 更新于 2016-07-04 | 分类于 Gossip | |
1,021 | 3

待历经沧海，待阅尽悲欢，心方倦知返。君已尘满面，污泥满身，好个迷途人。今日归来不晚，彩霞濯满天，明月作烛台。

这周上了两天班，出了两天差，见了三天朋友，现在是累个半死。可能是做事习惯的不同，工作是有些慢热的，总觉得每天为了不同部门的需求分散精力到处弄开口打补丁就是在给未来挖坑，未来的确是光明的，但是在坑里看，再光明也难受呀。

周三上午开完会把本周能做完的工作收尾一下，就和七牛的小伙伴往北京赶，结果到了机场托运行李的时候被告知航班取消，当晚就只有两班国航的飞机可以飞，一如既往没挤上，于是只好打道回府，明天早上再来。

塞翁失马，正好借着这个机会跟糗百的技术负责人好好学习了一路，从技术选型到如何推动团队，学到的不仅是做事情的方法，更是思考问题的角度。千辛万苦赶到了发布会会场，终于体验了一把一下飞机就赶趟的感觉。在发布会现场也和不同公司的人聊了聊直播，还真的是要深入进去才能看到个中门道，并且越深入就越人性，有的时候也就越不堪。

周五空了下来，去了趟首都博物馆，逛了下中国矿业大学，看了惊天魔盗团2，又见了下当年在微软时认识的小伙伴，该结婚的结婚，该带娃的带娃，一年一会，真的感觉到岁月产生的变化。但令我心暖的是，感情却没有随着时间流逝而褪色，反而愈发浓厚，言有尽而意无穷，说的就是那一杯杯以茶代酒吧。

周六早上等着吃凉皮的时候逛了一圈北京语言大学，第二次看到逸夫楼，配合上孔子学院的宣传栏，真切感受到『输出』的巨大影响。逸夫楼是一种方式，写教材、开课程同样也是一种方式。诚然，工作和生活中还有很多困惑，能做的是带着所有的不确定性，在隐忍中让自己变得更强。思考是一切的开始，去跨界，去走出自己的路。

中午和认识许久的书友见面吃了个饭，认识差不多快五年，一直以书评会友，见面之后真的是感觉说不完的话，不同的行业不同的思路不同的做法。还真有一个合作机会，能把我一直以来的夙愿做成真正有价值的内容以全新的形式分享给大家，虽然现在已经忙不过来，但是还是期待能把这事儿推动下去。

感情与抉择，明显能意识到现实的重量。正直毕业季，大约倒数第二批我认识的同学走向社会，我能在校园和社会间游离的方式一方面更多，一方面更少了。要真正去做一些事情，去推动改变，而不是等待改变。

下午和同学一路聊到晚上，我们刚到匹兹堡便认识，第一次吃饭竟是在我回国前夕。当她说『本以为上次之后一辈子都见不着你』的时候，真的感觉到个体的渺小，一旦失散，便难以重聚。

虽没有临表涕零，却真是不知所言。感谢爸爸妈妈的唠叨和关爱，感谢朋友们百忙之中抽空陪我逛吃逛吃，感谢公司让我出去长长见识。滴水之恩，定当涌泉相报。

以苦难为船，以泪为帆，心似离弦箭。莫说天无涯，海无岸，纵然归程须万载。今日归来不晚，与故人重来，天真作少年。
