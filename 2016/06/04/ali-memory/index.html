念拳王阿里

发表于 2016-06-04 | 更新于 2016-06-05 | 分类于 Thinking | |
1,315 | 4

和天意吃饭的一些感悟，惊闻阿里逝世的一点缅怀，以及回国后的几项计划。

回国也有一段时间了，本想着先修养一阵子，结果似乎比在匹兹堡更忙。一边倒时差一边找朋友叙叙旧，之后马不停蹄回老家给爷爷过生日。因为要回广州办手续，之前计划的探亲和自驾游都只好取消，不过也好，总算是能够歇息一下了。老规矩，想到哪写哪儿了。

阿里

我对阿里了解不多，大概是属于看过几个视频 + 维基百科 + 拳王阿里电影的水平，更不靠谱的是电影我还是在飞机上看的。面对三个多小时的老电影，真的是强忍着看完的，看到最后唯一印象深刻的就是灵魂音乐（然而在国内还是挺小众的），所以接下来我的领悟不深，请大家原谅。

阿里之于我，一部分是灵魂音乐，另一部分就是反抗精神。灵乐按下不表，反抗一切被强加在自己身上的观念思想，并且用自己的努力证明给所有人看，倒是有点《悟空传》的意思：

  • 我要这天，再遮不住我眼，要这地，再埋不了我心，要这众生，都明白我意，要那诸佛，都烟消云散！
  • 若天压我，劈开那天，若地拘我，踏碎那地，我等生来自由身，谁敢高高在上？

阿里的拳法路数和其他拳手都不一样，用他自己的话说就是『像蜜蜂一般攻击，像蝴蝶一般滑动』。联想一下梅西踢球的方式，同样适用。阿里和足球有着神奇的联结，会去球王贝利的退役赛捧场的一幕，真是惺惺相惜。

最触动我的，反而是阿里的这句话：

我知道我要去哪里，我知道真实的世界，我不想做别人让我做的事，我想做自己。

将来肯定会有各种各样的妥协，希望自己能勿忘初心，坚持下去吧。

天意

此天意非彼天意，这里的天意其实是我今天中午去见的朋友。虽然是师弟，但每次我都能从他身上学到很多。时隔一年再见，他又成长了不少，能这样和朋友大口吃肉大碗喝酒聊聊人生，快哉。这次聊天，少了些豪言壮语，多了些忧国忧民。

从天意口中得知了不少目前大一大二同学的状况，纵向对比，情况不容乐观。无论是每况愈下的参赛作品质量，还是愈发僵化的思维，深感痛心，也更有紧迫感做点什么。这里固然有院系调整之后『大学院』的因素在，但是领导急功近利，一切以成绩和奖项为纲的思路，也给同学们施加了不好的影响。比方说翻来覆去用之前得过奖的项目参加各类比赛，到最后看起来光鲜亮丽，但却是饮鸩止渴，再无创新和动手能力。大学院的领导不可能了解不同专业的不同要求，往往简单把教学数值化，很难想象创意如何在这样的环境中生存。好在年轻人走点弯路也正常，希望他们能尽早睁开眼睛看这个世界和自己要走的路吧，我也会尽力帮一把的。

如此杀鸡取卵，明眼人都看得出来，冷饭炒了一年又一年，再不扭转这个趋势，恐怕前辈们辛辛苦苦建立起来的名声，很快就要消耗光了。当然名声这些是虚的，可一旦学生不再有睁眼看世界的思维和格局，怕是再无翻身之日。

计划

其实也不能算是计划，吹好听点叫『布局』，不好听点就是『随便想想』，毕竟这些为两三年后做的打算，在大多数人眼中都没多少意义。

  • 尝试推进一下和中大合作的课程，算是校友的一点心意，无论是教材还是教学，很多课程都和实际太脱节，越早改变越好
  • 借由 StuQ 平台尝试开一个众筹小班课程，整理讲义和设计实验，也为之后的教材编写做准备
  • 除了和出版社继续接触，把自编教材内容放到网上，一边写一边接收反馈，不闭门造车。也不指望凭这个赚钱，只要大家愿意看，我就很开心了
  • 学习一下 web 应用的简单开发，和并行架构的设计，本职工作还是要努力做到最好
  • 深入挖掘一下基础云设施和客户之间在垂直领域的解决方案，了解 to B 的业务，着手开始设计和做原型开发
  • 把机器学习课程自己的理解和笔记更新在博客上，也算是一个复习

简单来说就是四个字：立言、成事。

共勉。
