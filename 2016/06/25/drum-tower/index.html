第二周 - 鼓楼

发表于 2016-06-25 | 更新于 2016-06-25 | 分类于 Gossip | |

我是个沉默不语的，靠着墙壁晒太阳的过客。如果我有些倦意了，就让我在这里独自醒过。

这是完整工作的第一周，也是渐入佳境的一周。随着对周围同事和部门业务的熟悉，终于能够站在和大家差不多的角度来观察、思考和处理事务了，这是好事。但另一方面，由于我们组负责全公司的后台系统，所以常常是很多个项目并行开发，每个人都需要处理多个项目，这种情况是我不太适应的。在学校里我的策略是集中时间和精力针对不同科目各个击破，但是公司的业务每天都在运转，也不断会有新的需求和新的问题，并行是不可避免的，这也对自己提出了更高的要求，无论是在时间管理还是事务优先级的判断上都是如此。

另外一个和学校里非常不同的是衡量成绩的尺度，这么说有点抽象，举个实际点的例子。网上有这么一句打趣的话『学霸能考一百分是因为只有一百分』，而到了工作中，首先一百分这个限制不存在了，甚至连满分都不存在了，真的是『小孩子才分对错，成年人只看利弊』。所幸我一直不是一个特别有所谓『目标感』的人，或者说我的目标都过于抽象，所以无论我做什么，只要做好了，都是在朝着目标进发。

在学校里最大的收获一是朋友，二是扎实的基础；在广泛阅读中最大的收获一是眼界，二是独立思考的能力。现在的问题在于，底层的理解和高层的认知因为缺乏大量具体实践，是割裂的。好的想法像空中楼阁一样无法落地，虽然很多东西明白内在的原理，但是因为不够熟悉整套体系（现在已经很少有机会可以从零起步做东西了），往往有力使不出。光有内力没用，还是得先一板一眼把招式练好，最后才到达『无招胜有招』的境界。

所以说没有压力是不可能的，好在大家对我都比较宽容，还是给了我不少时间来学习，我也要慢慢转变自己的工作方式，虚心向前辈和老员工请教学习，努力自己的适应性更强，更快在项目中发挥应有的作用。因为之前面对压力的经验不足，我还在尝试各种各样的解压方式，比方说从公司跑步回家，或者一下班就去找同学打牙祭。跑步的时候足够放空，只需要关注每一次呼吸和步伐节奏，是很好的放松；而和同学吃饭聊天，能够把一天一周的压力都说出来，再交流下各自遇到的喜怒哀乐，是很好的疏导。

毕业之后愈发珍惜见面的机会，能够坐在一起吃顿饭，真的是很大很大的缘分。无论是老朋友还是新朋友，喝喝茶聊聊天，是特别好的事情，既可以尝试不同的餐馆，也交流了各自的想法，说不定就是新的旅途开始的注脚。

工作上也认识了不少新朋友，第一次和同事打篮球，第二天在办公室见面就感觉亲切了许多，除此之外因为业务往来的缘故，还认识了七牛和 AWS 的朋友，他们身上有很多我可以学习的地方，希望自己能够在这段时间快速成长起来。

炎热的夏天在早上七点半准时到来，我的一天随着生物钟大概六点准时开启，和所有上班族一样赶车挤车打卡上班，但是心中怀揣的梦，还是很努力想守护好。

快活的人不眠，唯有我倚着围栏，对过往说晚安，晚安。
