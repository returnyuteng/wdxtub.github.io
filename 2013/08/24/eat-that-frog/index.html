【吃掉那只青蛙】梳理生活

发表于 2013-08-24   |   分类于 Reading   |     |  

也是一本如何做选择的书

昨天跟诚哥讨论拖延的问题，他说拖延和害羞，靠方法是没有办法根治的，得靠信念。所以问题就变成，我们要如何拥有这种信念。答案很简单，就是找到自己的梦想，然后拼了命也要一路追寻。而追寻的过程中，我们常常陷入瞎忙的陷阱中，这本书会教你一些方法，让你把宝贵的时间，留给最重要的事情。

这本书在豆瓣上的评价并不算太高，究其根本可能是因为国人在遇到自己没有的东西时，总是会预先设定成看不顺眼的模式。当然，理论为主确实是这本书的特点，但是这样才能让我们更好地发展出自己的一套方法不是吗？也许应试教育已经让大多数人习惯了标准答案，却丧失了探索未知的能力。

接下来会记下书中提到的所有我认为有用的东西，一来是读书笔记，二来与诸君共勉。

规则：如果你必须吃掉两只青蛙，那么要先吃那只长得更丑陋的。选重要的，说做就做，一做到底。

我想每个人身边都有这样的一类人，他们有很多想法，今天想做这个，明天想做那个，但是他们就是想想而已，却不去做。“只说不做”是最大的陋习，如果你发现自己有这个问题，那我建议你一定要时时刻刻提醒自己要立即采取行动。

规则： 如果你必须吃掉一只活青蛙，俺么即使你一只坐在那里并盯着它看，也无济于事。每天起来直接做最重要的事，并养成习惯。

很多时候我们拖延，是因为我们觉得一个事情难办，或者担心做不好。与其临渊羡鱼，不如退而结网，最重要的事情就是让你去做的，而不是去逃避的，忘掉自己的拖延症，你要做的就是把对你人生最有意义的事情先做完！

要养成良好习惯，3个D很重要：决心（Decision）、自律（Discipline）和坚定不移（Determination）。如果是男生，那么想要成为武士，3D是必不可少的；如果是女生，没有36D，3D也一样可以让你的生活精彩。

规则：最糟糕的利用时间的方法之一，就是把根本不需要做的事情做得很圆满。

我们要如何确定什么是最需要做的事情？明确目标！越清楚自己想要什么，越清楚如何才能实现自己的目标，就越容易克服拖沓并完成你的工作。这里有一个设定目标的方法：

  1. 确定自己究竟想要什么（这一步非常关键，请先想想你做某个事情的目的到底是什么，想清楚再继续）
  2. 把你的目标写下来（不要觉得你的记忆能战胜烂笔头，写下来的同时，也是一种提醒）
  3. 为你的目标设定一个完成的最后期限（一个事情有了起始和终止，这之间的距离才可以用努力来跨越，即使是长期计划，我也建议要给自己施加一些压力）
  4. 把视线目标需要做的事情都列出来（这实际上是审视自己目标的绝佳时机，从这里你应该能够明白如何能够把事情做完）
  5. 对清单进行整理，分轻重缓急（还是之前提到的，重要的事情先做）
  6. 立即根据计划采取行动（计划在纸上，但更应该在行动中）
  7. 下定决心，每天做一些能够接近目标的事情（多做一件事情，多努力一秒钟，你都会离目标更近）

最后送大家一个6P原则：Proper Prior Preparation Prevents Poor Performance.

规则：处理事情的时候，一定要抵制住先易后难的诱惑

80/20法则（帕累托法则）大家都知道，做20%的事情有80%的效果，即使会有些困难，但这又算什么伤？下定决心，从今天开始，把更多的时间花费在那些能让你不同凡响的少数工作上。

规则：制定长远目标有助于你确定自己的短期决策

规则：长远目标影响并常常决定你目前的所作所为

规则：你永远没有足够的时间去完成你要做的所有事情

一个人的成就与他在某一领域付出的努力成正比。你对待时间的态度，对你的行为和决定有着重大影响。

成功者是那些愿意推迟享受，做出短期牺牲的人。失败者总是只想缓解自己压力，而成功者则致力于实现自己的目标。如果你发现自己常常有“不如先放一阵子”的想法，那么你可能在滑向失败者的深渊。

经常问自己3个问题：

  1. 对我而言什么是最有价值的事情
  2. 什么事情只能由我来做，而且关系重大
  3. 此时此刻，怎样利用我的时间最有效。

一旦你能意识到每分每秒都是在为自己而活，并且真的如此做的话，就不会太愿意浪费时间的。

规则：只有不再去做那些无足轻重的事情，你的时间和生活才尽在掌握之中

如果你觉得别人让你做的事情并不是那么重要也不是非你不可，那么要学会拒绝，不要让原本在别人背上的猴子爬到你身上来。

这里推荐大家ABCDE法，你可以用这个方法给每天要做的事情分类，然后赶紧把重要的都做了：

  • A：非常重要，必须要做，不做后果严重
  • B：应该做，不做的话也不会带来严重问题
  • C：令人愉快，但做不做无所谓的事情
  • D：可以委派给别人做的事情
  • E：完全可以不做的事情

规则：你工作中最薄弱的那个环节，将决定你发挥自己技能和能力的限度。

每一项工作都可以分解为5~7个环节。找出自己工作的关键环节所在以及自己的优势，确保万无一失。

规则：在公司里，时间的质量最关键；在家里，时间的长短最重要。

这个规则翻译一下就是，工作的时候效率越高越好，效率高，时间就更有质量；放松的时候越轻松越好，尤其是要珍惜和家人或者亲近的人在一起的时间。

规则：在任何一个领域，要想获得成功，都要保持学无止境的精神

不要停止学习，学得越多你对自己就越有自信，对工作也就越胸有成竹。一旦你觉得自己能够做好一件事情，很可能就不会再拖沓，而是立即去做，然后证明自己的能力。这里还有两个克服拖沓的小技巧，第一个是在开始做事之前把一切都准备就绪，第二个就是一次只处理力所能及的一部分。

每个人都与众不同，有着自己特殊的天赋和才能。你要做的就是找出自己的独特之处，然后努力在这方面做到最好。成功人士就是那些肯花时间来考虑自己最擅长做什么，最喜欢做什么的人，说白了也就是扬长避短。

在显示与目标之前，存在着一个瓶颈。找出这个瓶颈并专心致志地去解决，通常能在更短的时间里取得更大的进展。想想这两个问题：

  1. 是什么制约着我实现自己的目标？
  2. 我为什么还没有实现自己的目标？

立即采取行动，什么都行，但是必须马上就动手！我们的命运都掌握在自己手中，必须学会给自己加压，否则就会再等待中虚度光阴。通过给自己加压，你的工作效率会更高，工作量也会大大增加。你会成为一个表现出色、成就不凡的人。

保持生活快乐、工作高效的一个和总要因素就是，让自己时刻精力充沛。你要知道自己什么时候状态最好，然后利用这些时间来处理最重要、最具挑战性的工作。只要饮食合理，营养均衡，定期锻炼，你就能更快、更好地完成自己的工作。

决定我们感受的不是我们遇上了什么事情，而是我们以什么心态去面对这个事情，乐观主义者有四个特征，看看在你身上有没有？

  1. 寻求事情好的一面
  2. 从挫折和苦难中总结经验
  3. 极力找出每个问题的解决办法
  4. 一言一行不离开他们的目标

要想保持沉着冷静、头脑清醒、做事搞笑，就有必要远离某些现代化通讯设备。不要成为高科技的奴隶，没有必要保持随时联系。试着在真的专心工作的时候把手机什么的放远一点，除了电话，其他都不要理会，来看看自己的效率到底有多高。

一次只做一项大工程的一小部分，要比从整体入手容易。成功而快乐的人的共性之一是，他们总是说做就做。他们只要听说一个好主意，马上会将它付诸实施，以证明这个主意对他们是否有帮助。

能否找到大块可利用的时间，关系到我们能否在工作中、生活中取得大的成就。可以把时间安排成一个固定的时间段，然后按照这些时间段来处理重要工作。安排特定时间段，安排日程表，抓住每一分钟。

一旦开始做一件事情，就应全力以赴，决不受外界干扰，也决不转移自己的注意力，直到百分百完成。

这本书的内容基本就这么多，说来说去其实一句话就可以总结了：

找到自己想做的事情，立即去做，一做到底。
