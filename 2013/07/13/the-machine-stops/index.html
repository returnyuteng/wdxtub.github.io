【大机器停止】牢笼

发表于 2013-07-13 | 更新于 2015-07-08 | 分类于 Reading | |
865 | 3

惊人的预见力，短且有力

这是一本不到百页的科幻小品，讲述的是人类后工业文明时候的事儿。除了字里行间流露出来的古典措辞之外，你很难相信这是一百多年前的作品。你不仅可以看到《黑客帝国》中机械时代的雏形，也能看到《黑镜》中科技极度发达后人类的生活，更可以看到《一九八四》的独裁与反抗。

我想在人类的内心深处还是抱有乌托邦的幻想的，虽然已经提出了近500年，但是似乎乌托邦的思想根基有着严重的问题，基本上来说它要求的是人类完全理性的生活，所以如果我们继续想的话，就可以得出这样的一个机器：控制着人们生活的方方面面，能提供的服务范围中可以随意获得，但是不允许做出系统禁止的行为，也不能提出任何异议。没错，这就是“大机器”。

但是乌托邦看起来是那么美好，以致于人们有意无意忽略了追重要的东西：人不可能是完全理性的。总会有人想要做一些大机器无法预见的事情，好奇和探索的精神永远深植于有些人心中，故事中的男主角就是这样。幸运的是，他熬到了大机器停止的时候，也见证了人类的灭亡

可以想象出人们在制造大机器的热情和动力，大家幻想着未来物质极度丰富的新世界，却没有意识到，其实他们正在建造自己的牢笼。关于这个我在讨论“慢网”的文章中也有提到，虽然现在还不明显，但是我们已经开始成为科技的奴隶了。你想一想上一次你离开电子设备是什么时候？隔一段时间不看看手机是不是有些难熬？你什么时候可以真正捧起书本，安安静静读完？

所以我想是时候要真正去放弃一些什么，例如不再一次又一次检查社交网络，不再发一条又一条的即时消息，我们真的需要以这样的密度保持联系吗？从现在的云和大数据，我们已经能够看到资源开始集中化了。且不说集中化对于世界政治的影响，就假设以后有一个世界国，我们日常的交流和工作都又统一的中枢处理，那么好，再进一步，首先体力劳动已经可以由机器人完成，然后每天所需的营养可以由空气提供，并且可以自由选择口味，气体会相应刺激大脑的不同部分，让你有真正在品尝美食的感觉，那么我们每天需要做什么？

所以要我说战争和大机器硬要选一个的话，我一定选择战争。如果要死于自己建造的牢笼中，那么我宁可在冲锋的路上倒下。

另外关于外星人入侵之后的新社会政治制度的讨论也很有意思，等我看完这一季的《陨落星辰》就再写一篇讨论这个话题的日志好了。
