电子阅读的字体们

发表于 2013-06-22   |   分类于 Gossip   |     |  

即使是字体，也要有自己的追求

今天看电子书的时候忽然有种异样的感觉，我想了一下，一定是字体的问题。很多时候为什么实体书看着舒服，排版固然重要，然而字体的作用也不可小视，大家可以随手拿起一本书，和电子设备对比一下字体的显示效果，大致就能发现问题所在。所以今天正好趁着这个机会，来聊聊用于电子阅读的字体们，为了说明我还是有一定资格来絮叨一下的，就先贴出我的阅历吧：

首先我要讲的是电子书和纸质书字体效果的不同，最重要的一点是渲染方式和显示方式的不同。纸质书的渲染部分实际上使用墨粉或者其他物理材料来实现的，几乎不会存在任何的锯齿问题，并且也不像屏幕会隔着一层玻璃，所以看起来会非常舒服；但是对于电子阅读尤其是在小屏幕设备上，渲染的问题实际上是非常难处理的，因为既然是屏幕，最小的组成部分就是像素，而即使在各种“视网膜”屏幕层出不穷的今天，字体的渲染依然是个问题。我想解决的方法除了是继续提高分辨率，还有就是设计更加优秀，更加适合屏幕显示的字体。

对于我们用户而言，如何选择适合自己的字体，就很重要了，在我看来一个好的适合电子阅读的字体至少需要符合以下几个条件：

  1. 应该是宋体或黑体的其中一种
  2. 字的结构要均衡
  3. 不能太黑或者太粗
  4. 在整版文字显示的时候，不能给人以压迫感
  5. 长时间阅读时不易疲劳

如果换成专业的标准，那么这些条件可以大致转换成这样：

  1. 笔画不应该有过多的边角修饰，边角宜平直
  2. 笔画应该尽可能横平竖直，减小渲染的复杂度
  3. 字体间距不宜过小
  4. 注意整体均衡，不宜过分锐利

那么如果我们根据这样的标准，来看看今天我要说的五种字体：

  1. 信黑体（比较新的高品质黑体，针对电子阅读优化）
  2. 方正书宋（显示风格较为锐利，但不至于刺眼）
  3. 方正小标宋（极显眼的字体修饰，给人以压迫感）
  4. 方正兰亭黑（略粗，重量感十足）
  5. 方正兰亭刊黑（略细，窈窕淑女）

总体来看都是宋体和黑体，楷体和书法体以及其他的非主流字体均不在考虑范围之内，评判标准上面提过。

这里要说的是本文的字体显示效果均是在Kindle Paperwhite安装多看系统上截图和测试的，因为不同系统的渲染和显示机制的不同所造成的视觉误差并不在本文的考虑范围之内。本文中选取了柴静的《看见》中第三章的某一段文字进行测试。好，下面就开始了。

信黑体

信黑体是柯炽坚老师筹备三年的字体，目标就是做出最高品质适合屏幕的现代黑体。我觉得经常进行电子阅读的同学第一眼就能感受到这其中的精心设计，大家可以对照之前我提出的几条标准来评判一下。可以看第六行的“分”字，那一捺还带有一段水平部分，这是其特点之一，是向传统中国书法的致敬；并且横平竖直，撇的偏移角度也非常考究，例如倒数第二行的“承认”，两个字中向左撇的一部分基本上是非常统一的；再看第三行的“什么”，这两撇也有不同，因为在字体的组成成分来说这两个成分是不一样的，需要体现出差异。大家可以看看其他字体的显示，感受一下其中的差异。

这套字体的最大问题反而是由于其最大的优势所导致的，就是因为单字的设计成本过高（不只是钱的问题，还有时间的问题）导致的字库不全。但是我相信随着时间推移这个问题是可以完善的，终有一天可以摘下中文黑体的桂冠。

方正兰亭刊黑

方正兰亭刊黑在我看来是方正兰亭黑的一个继承版本，字体更加娟秀和窈窕，正如其名字所寓意的那样，实际上这个字体更加适用于报刊杂志等等。不过出人意料的是其在电子设备上的显示效果也非常好。而且这个字库相对于信黑体来说要全很多，但是大家可以看看之前分析的几个字，就可以看出里面设计元素的多寡。相对于兰亭黑和信黑体来说，刊黑更加细，不过这也可能会导致在分辨率不是那么高的屏幕上显示效果有偏差。

我不好说这两种哪个字体更加好，因为它们都足以称之为中文黑体最顶尖的字体之一，大家可以对比一下两张图，至于个人喜好就仁者见仁智者见智了。我的感觉是这两种字体都非常柔和且在密集排版时不刺眼也没有压迫感，而这正是后面几种字体所不具备的优势。

方正兰亭黑

前一阵子其实我一直在使用这个字体，但是很容易疲劳，因为字形更加粗，更加接近原本的黑体，在这个不算太密集的排版中已经可以感觉出一种压迫感了。并且因为字形的宽度问题，会感觉出细节上的锯齿感，字形较前两种更大，虽然更加充实，可能也因此产生了压迫感。

这个字体更加适合当小标题，我相信长时间阅读的话，这并不是特别好的选择。

方正小标宋

小标宋是特别有意思的一个字体，看到这个我就很自然联想到了小时候经常翻看的新华字典，那里面的首字就用的是这种类型的字体。乍看起来感觉非常充实和华丽，笔画粗细不一，还有丰富的边角修饰，看起来就很官方。但是这样带来的问题同样明显，还是压迫感，这并不是一个特别适合长时间阅读的字体，但是不能否认这个字体真的非常漂亮。

方正书宋

这个字体我感觉是最适合电子阅读的宋体，首先字形较大，笔画的粗细变化没有那么明显。因为笔画较细致的缘故，所以边角修饰也不会那么突兀。但是这个字体依然有其问题，就是标点符号。大家可以看看图片中的各种问号，反正我是有种违和感，一个是问号破坏了整句话的平衡，没有占据字形的正中，二是风格不对，一看就感觉是两种字体。如果返回去看看信黑体的符号设计，就能明白其中的差异了。

总结

其实字体这个事儿，个人喜好在其中占据的比重特别大，纵使我说出千万的好，也抵不过一句“我就喜欢”。所以问题其实就在于能否找到更加适合自己的使用场景和阅读习惯的字体，例如平日的新闻或者是杂志阅读，那么有压迫感和充实感的字体的缺陷，其实反而会成为一种优势。

不同的字体其实会传达出不同的精气神，也适合不同的展现方式，我想如果能够更加好地运用字体和排版，那么电子阅读的体验一定会更好。

最后说说我的个人喜好，我其实很喜欢信黑体，当然刊黑也不错，宋体因为边角修饰的缘故，我反而不是特别喜欢，所以我的推荐度大概是这样的：

信黑 > 刊黑 > 书宋 > 兰亭黑 > 小标宋

这次就到这儿吧，字体到头来还是为了更好地阅读，所以多看看多想想，才是硬道理。
